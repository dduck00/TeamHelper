
# TeamHelper

## 0. 개요

    협업지원 서비스

## 1. 개발환경
    * Windows 10
    * JAVA "11.0.5" 2019-10-15 LTS
    * IntelliJ

----

## 2. 주요 기능
    * 그룹에 가입할 수 있음
    * 그룹 그림판
    * 그룹 채팅방
    * 그룹 일정 관리
    * 그룹 파일 공유
